# Ega Herawati

Personal portfolio website showcasing projects, skills, and experience in **Fullstack Software Engineering** with a focus on **technology-driven healthcare solutions**.

Built with a modern tech stack using **Next.js, TypeScript, and Tailwind CSS**.

---

## Live Demo



---

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* React Hook Form
* Zod
* Lucide Icons
* React Hot Toast

---

## Features

* Modern responsive portfolio design
* Project showcase with tech stack preview
* Experience timeline
* Certificate section
* Contact form with validation
* Downloadable CV
* Mobile responsive navigation

---

## Getting Started

Clone the repository

```
git clone https://github.com/egaherawati10/portfolio.git
```

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```

Open in browser

```
http://localhost:3000
```

---

## Build for Production

```
npm run build
```

Run production server

```
npm run start
```
